Em sexta-feira, 18 de março de 2016 16:17:09 UTC-3, Henrique Nogueira escreveu:
Não sei se é a melhor solução, mas vai la:
  • Cria uma view em /get_details/<id_do_pai>/ que vai retornar um json com informações do filho (dá uma olhada nos serializers do Django)
  • No seu template, use jQuery para faciliar:
$("#ID_DO_CHECKBOX_DO_PAI").change(function() {
    $.get('/get_details/<id_do_pai>/', function (response) {
        // Aqui vc processa o json que veio da sua view e popula a div com resultados, por exemplo:
        $('#ID_DA_DIV_ONDE_VAO_OS_FILHOS').html(
            <input type='checkbox' ...>
         );
    })
});

para controlar os ID's dos pais e filhos, vc pode usar custom HTML tags, como, por exemplo, data-target. 

Espero que ajude.
