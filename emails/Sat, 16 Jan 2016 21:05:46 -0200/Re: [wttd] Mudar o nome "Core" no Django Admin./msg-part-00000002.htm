Adriano você precisa editar um arquivo python chamado apps.py, caso não exista é só criar o arquivo na pasta core, dentro desse arquivo, você altera o nome, exemplo:

from django.apps import AppConfig


class CoreConfig(AppConfig):
    name = 'eventex.core'
    verbose_name = 'Outra coisa'
