Com a função fica fácil de gerar isso pra qualquer dos links.

No próprio link que exibe o modal eu chamo a função, utilizo data attributes pra deixar todos os dados que preciso já no link.

Um exemplo:
var data = {
    materia: $(this).data('materia'),
    ano: $(this).data('ano'),
    semestre: $(this).data('semestre'),
    ead: $(this).data('ead')
};

asyncModal(
    $(this).data('materia'),
    '/api/aluno/detalhemateria/html?{0}'.format($.param(data)),
    $(this).parent()
);
