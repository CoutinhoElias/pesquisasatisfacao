O campo 'slug' não está no form, mas ele é obrigatório.

O form que estou preenchendo tem o campo 'username'.
Dai eu queria que o slug fosse preenchido automaticamente a partir do username.
Eu tentei o código abaixo, mas sem sucesso. Ele salva um slug antigo armazenado em cache, eu acho.


class EmployeeForm(forms.ModelForm):
    password = forms.CharField(label='Senha', widget=forms.PasswordInput)

    class Meta:
        model = User
        fields = ('username', 'email', 'password')

    def clean(self):
        self.cleaned_data = super().clean()

        if self.cleaned_data.get('username'):
            self.slug = self.cleaned_data.get('username')

        return self.cleaned_data
