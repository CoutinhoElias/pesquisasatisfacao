==================================

angular.element

  1. - function in module ng

Retorna um elemento DOM nativo ou uma string HTML como um elemento  jQuery.

Se o jquery estiver disponível, angular.element é um sinônimo pra função jQuery. Senão, angular.element vai delegar pra implementação anguleira de um subconjunto do jquery, chamada "jQuery lite" or "jqLite."

jqLite is a tiny, API-compatible subset of jQuery that allows Angular to manipulate the DOM in a cross-browser compatible way. O jqLite implementa apenas as coisas mais comumente necessárias a fim de pesar pouco na sua página.

Pra usar o jQuery, basta garantir que ele seja incluído na página antes do angular.js.

Note: Todas as referências pra elementos no Angular são sempre objetos do JQuery ou do jqLite; nunca elementos DOM nativos.
