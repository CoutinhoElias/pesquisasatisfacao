Só explicando o que acontece aqui:

ng-repeat="value in dados | orderBy : ['categoria', '-nome'] : true"

  • O orderBy é um filtro angular
  • Eu sei disso pq eu chamo ele com "|"
  • Um filtro como esse é chamado assim: function orderBy(array_in, order, reverse){/*retorna outro array*/}
  • O array_in é o que está à esquerda do "|" (no caso, dados)
  • Os parâmetros seguintes são o que vem depois dos ":" order = ['categoria', '-nome'], reverse = true
  • O filtro orderBy é esperto o suficiente pra se virar caso order seja uma string ou um array de strings
  • Cada string pode ser precedida por "-" pra fazer um reverse apenas nela (eu não sabia disso!).
  • O parametro reverse=true no final troca a ordem de tudo (o default é undefined, que é considerado false)
