Sou apaixonado por Python e uso sempre que posso, pois além de simples, ela é muito eficiente e a performance é quase que comparável a linguagens compiladas e em determinadas situações, até mais performáticas que elas. Um exemplo disso é um script que utilizamos como uma function no Postgres (Sim, é possível usar Python com Postgres), pois a linguagem PSQL mesmo sendo eficiente, deixa a desejar quando existe um processamento grande de dados, mas o Python faz com brincando, tanto que a mesma function em PSQL leva 5 minutos para processar as informações de mais de 1 milhão de registro, o Python só leva 18 segundos para o mesmo conjunto de registros (Claro que poderíamos fazer um módulo em C++, mas nos testes com processamento de arrays a linguagem compilada só ficava 1 segundo mais rápida, então perda de tempo em ficar se criando módulos compilados e chatos, Python é mais simples e elegante).

Bom, continuando, mesmo não trabalhando com desenvolvimento Web, mas utilizamos muita a parte web para controle dos nossos softwares, que vão desde atualização automática, monitoramento, gerenciamento de licença e coleta de dados para analise pelos setores, o qual é feita em Django (Framework Python). Bom como os ajuste e melhorais deles (Infelizmente é uma das poucas coisa que ainda coloco a mão para implementar), acabei esbarrando com o AngularJS e fique apaixonado pelo que ele faz, ou seja, tornar a minha vida e do pessoal mais fácil (Igual ao Django) e com isso acabei chegando ao ng-master.
