no seu arquivo settings.py , você deve criar o MEDIA_ROOT [1][2]. Basicamente, é só você configurar o MEDIA_ROOT para algo como:

MEDIA_ROOT = os.path.join(BASE_DIR, 'mediafiles')


Dito isto, o MEDIA_ROOT deve ser uma pasta diferente de STATIC_ROOT

Adicionalmente, o parâmetro upload_to também aceita uma string que contem um formatador de data e hora no padrão strftime [3] ou uma função python [4].

Então no seu exemplo, você poderia definir para o django criar uma pasta dentro do diretório indicado por MEDIA_ROOT, como o nome "photos" e dentro dela criar outras pastas com o padrão ano/mes/dia/ e colocar o arquivo dentro: Fica assim:

photo = models.ImageField(upload_to='photos/%Y/%m/%d/')

[2] https://docs.djangoproject.com/en/1.9/howto/static-files/#serving-files-uploaded-by-a-user-during-development
[3] https://docs.python.org/3/library/time.html#time.strftime
[4] https://docs.djangoproject.com/en/1.9/ref/models/fields/#django.db.models.FileField.upload_to
