O que eu fiz depois de clonar o projeto do github foi o seguinte:

1) pelo terminal estar na pasta raiz do projeto onde fica o .gitignore
2) o heroku tem um comando heroku git:remote
segue abaixo o help dele:
$ heroku git:remote --help
Usage: heroku git:remote

adds a git remote to an app repo

 -a, --app APP       # the Heroku app to use
 -r, --remote REMOTE # the git remote to create
 --ssh-git           # use SSH git protocol

extra arguments will be passed to git remote add

Examples:

  $ heroku git:remote -a example set git remote heroku to https://git.heroku.com/example.git

3) ae basta pegar o nome da app no heroku o comando é:
$ heroku apps

4) Ae ja temos td é so rodar:
 heroku git:remote -a NOME_DA_APP_NO_HEROKU

5) para verificar roda isso aqui:
$ git remote -v

ae vai mostrar esse output aki parecido com o meu:
heroku    https://git.heroku.com/eventex-rtancman.git (fetch)
heroku    https://git.heroku.com/eventex-rtancman.git (push)
origin    git@github.com:rtancman/eventex.git (fetch)
origin    git@github.com:rtancman/eventex.git (push)
