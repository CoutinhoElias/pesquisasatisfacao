Escrevendo a função every() do JS que retorna um boolean de acordo com todos os items do array aplicados a uma função em específico (Python escolhido)

>>> every = lambda array, fn: all([item for item in map(fn, array)])
>>> all_digits = lambda array: every(array, str.isdigit)
>>> l1 = ['1', '2', '8', '678']
>>> all_digits(l1)
True
>>> l2 = ['1', '2', '8', '678', 'a']
>>> all_digits(l2)
False
