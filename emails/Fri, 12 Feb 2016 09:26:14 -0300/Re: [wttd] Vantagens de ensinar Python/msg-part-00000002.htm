Com Python pode-se ter mais foco na logica de programação do que ficar brigando com sintaxe de linguagem, o que é muito bom para um curso introdutório.

Dito isto,a Python tem algumas funcionalidades na linguagem que são de muito alto nível e que, na minha opinião, para o objetivo de ensino como primeira linguagem pode não ser tão bom para a aprendizagem.

Para este problema, o professor pode utilizar o livro "Introdução à Programação com Python" do Nilo Ney Coutinho Menezes ( https://twitter.com/lskbr ), como livro de referencia para a primeira linguagem de programação.

No livro, ele propositalmente omite e deixa de utilizar alguns recursos da linguagem Python para poder focar mais no ensino do básico de programação que servirá para qualquer outra linguagem:

http://www.novatec.com.br/livros/introducao-python-2ed/
